# _Rock Paper Scissors Game_

#### _Rock Paper Scissors Game, 4/25/18_

#### By _**Eva Antipina**_

## Description

_A web app that let two people play a classic game Rock Paper Scissors. Each player should choose rock, paper or scissors in a checkbox field and submit. The program will tell who is the winner respectively to the rules. If one person chooses "Rock" and the other person picks "Scissors" then "Rock" wins. If one person chooses "Paper" and the other person picks "Scissors" then "Scissors" wins. If one person chooses "Rock" and the other person picks "Paper" then "Paper" wins. It's a game of combinations._   

## Specifications

* _The program returns message "Player 2 wins!" if Player 1 chooses "Rock" and Player chooses "Paper"._
  * _Example Input: "Rock", "Paper"_
  * _Example Output: ["Player 2 wins!"]_

* _The program returns message "Player 1 wins!" if Player 1 chooses "Paper" and Player chooses "Rock"._
  * _Example Input: "Paper", "Rock"_
  * _Example Output: ["Player 1 wins!"]_

* _The program returns message "Player 1 wins!" if Player 1 chooses "Rock" and Player chooses "Scissors"._
  * _Example Input: "Rock", "Scissors"_
  * _Example Output: ["Player 1 wins!"]_

* _The program returns message "Player 2 wins!" if Player 1 chooses "Paper" and Player chooses "Scissors"._
  * _Example Input: "Paper", "Scissors"_
  * _Example Output: ["Player 2 wins!"]_

* _The program returns message "Draw" if Player 1 chooses "Paper" and Player chooses "Paper"._
  * _Example Input: "Paper", "Paper"_
  * _Example Output: ["Draw"]_

## Setup/Installation Requirements

* _Clone or download the repository._
* _Unzip the files into a single directory._
* _Open the Windows PowerShell and move to the directory_
* _Run "dotnet restore" command in the PowerShell._
* _Run "dotnet build" command in the PowerShell._
* _Run "dotnet run" command in the PowerShell._
* _Open a web browser of choice._
* _Enter "localhost:5000/" into the address bar._
